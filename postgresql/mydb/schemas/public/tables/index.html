7Ô∏è‚É£ Triggers (automatic behavior)#

Triggers are actions that fire:

  • BEFORE INSERT

  • AFTER UPDATE

  • BEFORE DELETE

Example:

BEFORE UPDATE ON users
FOR EACH ROW
EXECUTE FUNCTION set_updated_at();
Used for:

  • Audit fields

  • Validation

  • Business rules
